# Contributing Guidelines

Thank you for helping make this list awesome!

## Adding a new resource

Please ensure your suggestion meets these criteria:

- **Terraform/OpenTofu relevant** — The tool, framework, or resource must be usable
  with or directly relevant to Terraform/OpenTofu workflows.
- **Actively maintained** — The project should have meaningful activity within the
  last 12 months. Archived or abandoned projects will not be accepted.
- **Minimum maturity** — Open-source tools should have at least 20 GitHub stars and
  be at least 30 days old.
- **Genuinely useful** — Only submit resources you have personally used or thoroughly
  evaluated. This is a curation, not a collection.
- **Not marketing** — Descriptions must be factual. Avoid taglines, superlatives,
  or promotional language.

## Format

Use this exact format for new entries:

```
- [Name](https://example.com/) - Short, factual description ending with a period.
```

- Start the description with an uppercase letter.
- End with a period.
- Use a dash (` - `) to separate the link from the description.
- Add ` 💲` after the description for commercial/paid products.
- Add ` 🆓` after `💲` if a free tier is available.
- Add ` 🇪🇺` for European regulation-specific resources.
- Add ` 🏛️` for government/public sector-focused resources.
- Add ` ⚠️` for archived/deprecated resources (only if they are well-known).

## Process

1. Fork this repository.
2. Add your entry in alphabetical order within its section.
3. Ensure the entry follows the format above.
4. Submit a pull request with a clear description of what you're adding and why.
5. One entry per pull request (makes review easier).

## Updating or removing entries

- If a project has been archived or abandoned, open an issue.
- If a link is broken, open a PR with the fix or an issue.

## Commercial products

Commercial products are welcome if they provide genuine value to the
Terraform compliance ecosystem. They must be marked with 💲 and described
factually. Products with free tiers should also include 🆓.
